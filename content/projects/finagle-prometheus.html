--- title: finagle-prometheus ---

finagle-prometheus is a Scala library that exposes metrics from your Finagle application to Prometheus.

There are two parts to it. The first is a component which reads Finagle's internal metrics, and converts these to the equivalent classes from Prometheus' simpleclient library. The second is a selection of Filter classes that can be added to clients, to expose additional custom metrics for those clients. These are currently focussed on HTTP clients.

Crucially, it also provides a handler that you can mount to an HTTP endpoint for Prometheus to scrape.

The GitHub repository includes an example of how to use all of these things together.
